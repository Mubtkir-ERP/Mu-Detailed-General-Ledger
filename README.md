# Mu Detailed General Ledger

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Frappe](https://img.shields.io/badge/Frappe-v15-blue)](https://frappeframework.com)

A comprehensive, line-by-line General Ledger report for ERPNext that displays every transaction detail instead of aggregated totals.

تقرير شامل لدفتر الأستاذ يعرض كل سطر من GL Entry بشكل منفصل بدلاً من الإجمالي.

## Key Features

✅ **Line-by-Line Display**: Shows every GL Entry line separately  
✅ **Comprehensive Filters**: All standard filters  
✅ **Running Balance**: Automatic cumulative balance calculation  
✅ **Accounting Dimensions**: Full support  
✅ **Custom Fields**: Automatically detected  
✅ **Color Coding**: Visual differentiation  

## Installation

```bash
bench get-app https://github.com/YOUR_USERNAME/mu_detailed_gl_report
bench --site your-site install-app mu_detailed_gl_report
bench --site your-site migrate
```

## Usage

Navigate to: **Accounting > Reports > Detailed General Ledger**
